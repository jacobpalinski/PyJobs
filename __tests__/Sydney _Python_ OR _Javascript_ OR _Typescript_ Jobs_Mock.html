          Discuss, analyze, and understand new feature requirements with your team lead and PM


          Overview


          At the company, we are looking for a front-end developer that will help us build new products in the blockchain space and help us add new features for some of our current projects, like some new stealth projects that we are starting to work on.


          Responsibilities


          • Discuss, analyze, and understand new feature requirements with your team lead and PM.
          • Work directly with designers and engineers to implement new UI features, as well as improve the UX of existing features.
          • Work on integrating the app with the backend. Define requirements and improvements of the backend API.
          • Ensure that the code base and its main dependencies are always up-to-date.
          • Write unit/E2E tests, as well as perform manual tests.
          • Investigate bugs and provide support for users.
          • Participate in product meetings.


          Requirements


          • 0 to 2 years of experience in Software development
          • 1+ years of experience in React Native/React (preferable) and other web and mobile technology experience (breadth and depth)
          • Expertise with Typescript
          • Agility prototyping UIs and proficiency in writing clean JS/JSX and styling code.
          • Good understanding of API architecture following best practices.
          • Proficiency in the creation of tests (unit tests, integration, etc).
          • Basic knowledge of Linux
          • Good knowledge of Git
          • Good level of verbal and written communication


          Preferred Qualifications/Bonus Points


          • Knowledge of native mobile development (ie. Java for Android or Objective-C/Swift for iOS) and design patterns.
          • Experience with Xcode and development of software for Apple platforms.
          • User-level experience with blockchain applications (eg. wallets, dApps, staking, etc)
          • Understanding of blockchain fundamentals
          • React native web applications
